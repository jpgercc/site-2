HTML:


Como criar uma estrutura básica HTML:

Para iniciar o desenvolvimento front-end, precisa-se de um editor de textos ou um IDE (Integrated Development Environment)
Usando o app sublime, selecione uma pasta raiz para iniciar o projeto (File->Open Folder). Para mostrar ao Sublime qual linguagem você está usando = View->Syntex->HTML.
Após criar a primeira página do site, o nome dado ao arquivo deverá obrigatoriamente ser INDEX + . + a língua usada para programação, desta forma o servidor que carregar sua página saberá que esta deve ser a primeira página a ser carregada.
Como este projeto é feito com linguagem HTML, foi salvo na pasta raiz como index.html HTML é uma linguagem de marcação, então deve-se marcar o conteúdo para ser entendido pelo navegador através das TAGS, todas as TAGS têm aberturas e fechamentos. A partir do HTML-5 as TAGS passaram a ser semânticas.

CSS:


Formatação folha de estilo CSS:
